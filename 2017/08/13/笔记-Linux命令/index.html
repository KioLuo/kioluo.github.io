//系统编码设置(GBK、UTF-8)

locale查看编码设置

若不能运行locale-gen，可运行以下命令

apt-get purge locales

apt-get autoclean

apt-get install locales

修改默认中文编码字符：

vi /etc/profile

export LC_ALL=”zh_CN.GBK”

export LANG=”zh_CN.GBK”

export LANGUAGE=”zh_CN:zh:en_US:en”

或者vi /etc/environment

LANG=”zh_CN.GBK”

LANGUAGE=”zh_CN:zh:en_US:en”

添加字符编码：

  1. 直接使用locale-gen

locale-gen zh_CN.GBK

  1. 修改/var/lib/locales/supported.d/local文件

在文件尾添加中文字符集

zh_CN GBK

退出后保存

dpkg-reconfigure locales

重启

//用户和用户组管理

与用户相关的配置文件：

/etc/passwd 注：用户（user）的配置文件；

/etc/shadow 注：用户（user）影子口令文件；

与用户组相关的配置文件：

/etc/group 注：用户组（group）配置文件；

/etc/gshadow 注：用户组（group）的影子文件；

$ sudo useradd -g ddc [用户名] #--------------添加新用户

$ sudo passwd [用户名] #--------------给新用户创建密码

$ sudo usermod -g ddcuser [用户名] #--------------将新用户改为ddcuser组

#新用户修改自己的密码：

$ passwd

用户管理

  1. useradd //添加用户
  2. adduser //添加用户
  3. passwd //为用户设置密码
  4. usermod //修改用户信息
  5. pwcov //同步用户从/etc/passwd到/etc/shadow
  6. pwuncov //pwcov逆操作

用户组管理

  1. groupadd //添加用户组
  2. groupdel //删除用户组
  3. groupmod //修改用户组信息
  4. groups //显示用户所属用户组
  5. grpcov //同步用户组从/etc/group到/etc/gshadow

权限管理

  1. chmod //更改文件权限信息
  2. chown //更改文件拥有者
  3. chgrp //更改文件所属用户组
